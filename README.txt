I.Overview
This module is a teleport request function plug-in in the Minecraft Forge environment, 
allowing players to send transfer requests to other players in the game for convenient location movement.
II.Installation
1. Prerequisites
Make sure you have the appropriate version of Minecraft Forge installed. The version of Forge must be compatible with both the game version you're using and this teleport request mod.
2. Download
Obtain the.jar file of this mod from a trusted source. Be sure to download the version that matches your Minecraft and Forge setup.
3. File Placement
Navigate to your Minecraft game directory. Locate the "mods" folder within it. Copy and paste the downloaded.jar file into this "mods" folder.
4. Launch the Game
After placing the file, start your Minecraft game. The mod should be successfully installed and ready for use.
III.Usage
1.Send a Teleport Request: Open the in-game chat bar and enter /tpa [PlayerName].
2.Receive request: After receiving the request, enter confirmation or rejection instructions in the chat bar as prompted (e.g., /tpaccept [Sender] for acceptance, /tpdeny [Sender] for rejection).
